# App-Recommendation-System
An App Recommendation System using TF-IDF and Cosine Similarity on Google Play Store Data

## Overview
An end-to-end Machine Learning application that recommends Android apps based on user preferences. This project uses TF-IDF Vectorization and Cosine Similarity to find content-based relationships between apps using the Google Play Store dataset. 

## Features
- Data Cleaning Pipeline: Robust preprocessing to handle missing ratings, inconsistent sizes and duplicate entries. 
- Smart Filtering: Removes junk apps, test entries and low-popularity items to ensure high-quality recommendations.
- Content-Based Filtering: Uses categories and genres to build a high-dimensional feature space.
- Interactive UI: A sleek Streamlit interface for searching and discovering apps in real time.
